# üöÄ Quick Start Guide

Get up and running in 3 minutes!

## Step 1: Install Python
Make sure you have Python 3.8 or higher installed.

Check your version:
```bash
python --version
```

If you need Python, download from: https://www.python.org/downloads/

## Step 2: Clone or Download
**Option A - Clone with Git:**
```bash
git clone https://github.com/yourusername/income-strategy-engine.git
cd income-strategy-engine
```

**Option B - Download ZIP:**
1. Click the green "Code" button on GitHub
2. Select "Download ZIP"
3. Extract the ZIP file
4. Open terminal/command prompt in that folder

## Step 3: Install Dependencies
```bash
pip install -r requirements.txt
```

**Troubleshooting:**
- If `pip` doesn't work, try `pip3`
- On Windows, you might need to use `python -m pip install -r requirements.txt`
- On Mac/Linux, you might need `sudo pip install -r requirements.txt`

## Step 4: Run the App
```bash
streamlit run app.py
```

**Troubleshooting:**
- If `streamlit` command not found, try `python -m streamlit run app.py`
- Make sure you're in the correct folder (where app.py is located)

## Step 5: Open in Browser
The app should automatically open in your browser at:
```
http://localhost:8501
```

If it doesn't open automatically, copy that URL into your browser.
